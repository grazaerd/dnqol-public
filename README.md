# DNQOL
## Official clients and DX11 only.
> - Tested on SEA. 

## Note
The release is already overdue, this should have been released 5 months ago, but this is a "free-tier" mod, so don't expect too much.

## Disclaimer
**Use at your own risk, don't blame me, especially regarding getting banned or losing your raids (crashes, graphical errors etc.).**

## How to use it
```
└── Dragon Nest Folder
    ├── ddraw.dll
    ├── dragonnest_x64.exe
    └── dnconfig.ini
```
- **To open the GUI press ```- =``` on your keyboard**
- There will be a config named ``` dnconfig.ini ``` which you can modify if some features crashes the game. 
## List of features
- Camera
  - Camera Zoom
  - Camera Shake
  - Camera Animation
  - FOV
- Shader
  - Gray Screen on Dead
  - Outline
  - Weapon Trails
  - Color Post Process
- UI
  - Hit Combo
  - Total Damage
  - Pick Item on Chat
  - Guide Star Alert
- Misc
  - Window Ghosting
  - Set Process Thread to High
  - Process Priority
  - Bandicam Thread
  - Auto Skip Cutscene (Cutscene that needs to hit ESC)
  - FPS Limiter
