# DNQOL
## Official clients and DX11 only.
> - Tested on SEA. 

## Disclaimer
**Use at your own risk, especially getting banned or losing your raids. Don't blame me if you got banned or crash on-going raid**.


## List of features
- Camera
  - Camera Zoom
  - Camera Shake
  - Camera Animation
  - FOV
- Shader
  - Gray Screen on Dead
  - Outline
  - Weapon Trails
  - Color Post Process
- UI
  - Hit Combo
  - Total Damage
  - Pick Item on Chat
  - Guide Star Alert
- Misc
  - Window Ghosting
  - Set Process Thread to High
  - Process Priority
  - Bandicam Thread
  - FPS Limiter


## How to use it
```
└── Dragon Nest Folder
    ├── ddraw.dll
    └── dragonnest_x64.exe
```
- There will be a config named ``` dnconfig.ini ``` which you can modify if something is crashing while using the dll.
